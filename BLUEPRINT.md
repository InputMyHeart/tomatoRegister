# 番茄记账项目蓝图

## 项目定位

番茄记账是一款家庭式个人记账微信小程序，使用小程序云开发作为后端能力。产品重点不是重后台和复杂权限，而是让家庭成员能够轻松记录、查看、共享账本，并保持温暖、好用、辨识度强的番茄色 UI。

## 最高优先级约束

- 全局 UI 风格必须保持统一，继续沿用当前番茄色视觉体系。
- 不要随意更改颜色、背景、圆角、卡片语言、图标风格、navbar 风格，除非明确要求。
- `miniprogram/app.js` 中的 `globalData.env` 由用户手动维护，后续改动不得覆盖或清空。
- 云函数新增放在 `cloudfunctions` 下，不能改动 `quickstartFunctions`。
- 分析页暂时不动。

## 当前 UI 风格

- 主色：番茄红 `#F0442F`。
- 背景：温暖浅色背景，配合轻微番茄色渐变。
- 图标：统一使用 Remix Icon 线性风格，通过本地 `ri-icon` 组件渲染。
- Logo：使用 `miniprogram/images/brand/tomato-ledger-logo-256-transparent.png`。
- Navbar：自定义 navbar。
  - Tabbar 页面：保留 logo、番茄记账品牌名和中间页面标题。
  - 非 tabbar 页面：仅显示返回按钮和中间页面标题。

## 登录与用户资料

已完成真实微信云开发登录串联：

- 前端通过 `app.loginWithWechat()` 调用 `tomatoLedger` 云函数的 `login` action。
- 云函数通过 `cloud.getWXContext()` 获取真实 `OPENID`。
- 首次登录自动创建用户和默认“我家账本”。
- 用户唯一身份判断以 `openid` 为准，不依赖昵称。
- 默认昵称为 `番茄用户#xxxx`，四位数编号确保首次生成时不重复。
- 默认头像为小程序 logo。
- 默认性别为 `喵星人`。
- 资料编辑页支持头像、昵称、性别。
- 性别选项：`喵星人 / 男生 / 女生`。
- 用户可以保留自己的系统编号昵称，但不能改成他人的 `番茄用户#xxxx` 格式昵称。

## 我的页进度

已完成：

- 顶部用户信息卡片。
- 登录后整张用户信息卡片可进入编辑资料页。
- 用户卡片右侧设置图标作为编辑资料入口提示。
- 金刚区展示：使用天数、记录笔数、账本总数。
- 未登录时金刚区使用毛玻璃效果，仅提示“登录后查看”。
- 账本管理模块：创建账本、加入账本、账本配置。
- 记账设置模块：分类管理、支付账户、月预算、月度起始日。
- 其他模块：导入账单、导出账单、意见反馈、更新日志。
- 登录后底部展示退出账号按钮。

## 首页蓝图

已确认方向：

- 顶部账本名称需要有切换账本入口，可通过图标提示可切换。
- 首页不显示当前用户在账本中的角色。
- 首屏财务数字需要弱化，避免负结余或不符合预期时造成记录压力。
- 最近记录卡片移除 AI 感较强的副文案。
- “记一笔”是高频操作，应作为全局悬浮按钮组件存在。

## 明细页蓝图

已确认方向：

- 明细页按月份展示当前账本记录。
- 不做过多数据化，避免和分析页重叠。
- 支持切换账本查看明细。
- 支持全局快速记一笔入口。
- 按天分组，每天之间有清晰分割。
- 每天分割处展示当日收入/支出简要统计。
- 保留简单筛选能力。
- 长列表加入吸顶辅助。
- 缺省页面统一组件：图标 + “暂无数据”。

## 账本与权限蓝图

- 一个用户可以存在多个共享账本，也可以创建独立账本。
- 角色：创建者 / 成员。
- 核心操作仅创建者可操作。
- 轻量操作所有成员可操作。
- 共享账本中：创建者可以编辑/删除他人的记账，成员只能操作自己的记账。
- 只读分享仅可查看，不允许操作、导出、导入等行为。
- 邀请优先使用邀请码，同时保留微信分享卡片入口。

## 记账模型蓝图

- 收入/支出记录。
- 默认分类覆盖常见场景。
- 支持自定义新增分类。
- 保留支付账户字段。
- 支持预算功能。
- 支持月度起始日设置。

## 已实现技术点

- `cloudfunctions/tomatoLedger`：统一业务云函数。
- `login`：真实微信登录、创建用户、创建默认账本、返回用户与统计。
- `updateProfile`：保存头像、昵称、性别。
- `ensureCollections`：首次使用时尝试创建必要集合。
- `app.js`：统一登录态、资料更新、退出登录、本地缓存。
- `custom-navbar`：区分 tabbar 页面和普通页面视觉。
- `ri-icon`：本地 Remix Icon 字体组件。
- `empty-state`：统一缺省组件。
